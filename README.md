# MiniPlayer
Minim-based music player, beta stage.
Low CPU usage, high RAM usage.
Don't expect too much.

# Depedencies
* Minim
* ControlP5
* Java 8
* Processing 3

# How to work on it?
* 1. Clone this repo into your sketchbook
* 2. Open all files from Processing 3 GUI
